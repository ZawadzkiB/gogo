/gogo:skills runtime flow (as-built)

flowchart TD
  ARGS{"invocation"}:::gate
  ARGS -->|""extract X" (directed)"| TARGET["resolve the named section"]:::proc
  ARGS -->|"no prompt (audit)"| SCAN["scan .gogo/knowledge/*.md"]:::proc
  SCAN --> MEASURE["measure body lines\nOK<200 · WARN 200-400 · OVER>400"]:::proc
  MEASURE --> ANALYZE["analyze sections in WARN/OVER\n(cohesion · context-locality · standalone-able)"]:::proc
  ANALYZE --> CLASSIFY
  TARGET --> CLASSIFY["classify kind\nknowledge | standalone"]:::proc
  CLASSIFY --> PROPOSE[/"proposals: slug · kind · destination · description · lines saved · scripts/env · stub"/]:::art
  PROPOSE --> GATE{"user approves\n(per candidate)?"}:::gate
  GATE -->|"no / edit"| STOP([stop — nothing written]):::io
  GATE -->|"knowledge"| WK["write .gogo/skills/<slug>/SKILL.md\n+ scripts/ + .env.example"]:::out
  GATE -->|"standalone"| WC["write .claude/skills/<slug>/SKILL.md\n(harness auto-discovers)"]:::out
  WK --> STUB
  WC --> STUB["replace parent section\nwith summary + Load-when pointer"]:::out
  STUB --> INDEX["update .gogo/skills/index.md\n(kind · destination · trigger)"]:::out
  INDEX --> REMEASURE["re-measure parents\n(confirm under budget)"]:::proc
  REMEASURE --> REPORT([report: lines saved · skills created]):::io

Propose → approve → extract hand-off (as-built)

sequenceDiagram
  actor U as user
  participant S as /gogo:skills
  participant K as .gogo/knowledge/*.md
  participant GS as .gogo/skills/ (+ index.md)
  participant CS as .claude/skills/
  U->>S: /gogo:skills  (or "extract X")
  S->>K: audit body lines (OK/WARN/OVER) · classify candidates
  K-->>S: sections + kind (knowledge | standalone)
  S-->>U: proposals (slug · kind · destination · lines saved) — STOP
  U->>S: approve / override kind / decline (per candidate)
  alt approved knowledge
    S->>GS: write .gogo/skills/<slug>/SKILL.md (+ scripts/.env)
    S->>K: replace section with "Load when:" pointer
    S->>GS: update index.md registry
  else approved standalone
    S->>CS: write .claude/skills/<slug>/SKILL.md (only sanctioned write outside .gogo/)
    S->>K: replace section with "Load when:" pointer
    S->>GS: update index.md registry
  else declined
    S-->>U: nothing written
  end
  S->>K: re-measure parents (confirm under budget)
  S-->>U: report (lines saved · skills created)
