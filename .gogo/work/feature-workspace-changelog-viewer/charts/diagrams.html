gogo diagrams — workspace-changelog-viewer

Artifact flow (intended design)

flowchart TD
  WORK["work/feature-<slug>/ (plan · state · decisions · review · test)"]:::art
  WORK --> REPORT["report ⑤ (gogo-knowledge)"]:::proc
  REPORT --> RDIR[/"work/feature-<slug>/report/\nreport.md + UML (class·sequence·activity·use-case) + diagrams.html"/]:::art
  RDIR --> DONE{"/gogo:done\n(user: this is shipped)"}:::gate
  DONE -->|"copy"| CL[/".gogo/changelog/<date>-<slug>/\nreport.md + diagrams"/]:::art
  RES["resources/mermaid.min.js + interactive viewer (pan/zoom/drag)"]:::out
  CL --> VIEW["/gogo:view (gogo-view)"]:::proc
  RDIR --> VIEW
  RES --> VIEW
  VIEW --> PAGE([self-contained webpage: readable summary + interactive diagrams]):::io
