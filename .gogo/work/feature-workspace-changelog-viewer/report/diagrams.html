Artifact flow

flowchart TD
  WORK["work/feature-<slug>/ (plan · state · decisions · review · test)"]:::art
  WORK --> REPORT["report ⑤ — gogo-knowledge (strict | lenient)"]:::proc
  REPORT --> RDIR[/"work/feature-<slug>/report/\nreport.md + UML (.mmd) + diagrams.html + manifest.json"/]:::art
  RBROKEN["/gogo:report <feature> (past/broken run)"]:::proc --> REPORT
  RDIR --> DONE{"/gogo:done\n(report present?)"}:::gate
  DONE -->|"missing → STOP: run /gogo:report first"| STOP([stop]):::io
  DONE -->|"present → copy"| CL[/".gogo/changelog/<date>-<slug>/"/]:::art
  RES["resources/ — mermaid.min.js + viewer/ (pan·zoom·drag)"]:::out
  CL --> VIEW["/gogo:view (gogo-view)"]:::proc
  RDIR --> VIEW
  RES --> VIEW
  VIEW --> PAGE([offline page: readable summary + interactive diagrams]):::io

/gogo:view runtime (sequence)

sequenceDiagram
  actor U as user
  participant V as /gogo:view (gogo-view)
  participant CL as .gogo/changelog/ + work/*/report/
  participant RES as .gogo/resources/ (mermaid + viewer/)
  participant P as page (.gogo/resources/view/<name>.html)
  participant B as browser
  U->>V: /gogo:view  (or a slug/entry)
  V->>CL: enumerate selectable reports
  CL-->>V: changelog entries + in-progress work reports
  V-->>U: list → pick one
  U->>V: choose report
  V->>RES: ensure mermaid.min.js + viewer/{interactive.js,viewer.css}
  V->>V: pre-render report.md → HTML; inline each .mmd verbatim
  V->>P: write self-contained page (relative ../mermaid.min.js, ../viewer/*)
  V->>B: open (best-effort) — else print file:// path
  B->>B: render diagrams → wrap each in pan/zoom/drag canvas
  B-->>U: readable summary + interactive diagrams
