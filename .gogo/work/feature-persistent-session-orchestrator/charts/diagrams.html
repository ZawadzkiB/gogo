Two orchestrators over one skill — the runtime sequence

  actor User
  participant CLI as gogo CLI (session manager)
  participant Lock as lock + registry
  participant P as claude -p session (by UUID)
  participant Skill as /gogo:go skill

  User->>CLI: gogo go slug
  CLI->>Lock: acquire owner lock · resolve first-vs-resume UUID
  CLI->>P: claude -p '/gogo:go slug' (--session-id or --resume)
  P->>Skill: run implement in-context (warm)
  Skill->>Sub: Task(gogo-reviewer) / Task(gogo-tester) — fresh eyes
  Sub-->>Skill: findings → review/test issues.json
  Skill->>Skill: warm fix loop, then report ⑤
  P-->>CLI: process exit + JSON envelope (session_id, total_cost_usd)
  CLI->>Lock: record status (parked / awaiting-uat) · release lock
  CLI-->>User: surface awaiting-uat (run /gogo:done) or the parked gate
  Note over User,P: later leg — gogo go slug --resume recalls the SAME warm UUID (cost model: transcript re-sent)

Session lifecycle — states and reap

stateDiagram-v2
  [*] --> None
  None: No session tracked
  Running: Persistent claude -p session running the skill
  Parked: Exited at a decision gate (waiting-for-user)
  AwaitingUAT: Report green, waiting at the UAT gate
  Shipped: gogo done archived the entry
  Orphaned: Crash or lost owner, lock stale
  Reaped: Session and tmux killed
  None --> Running: gogo go or gogo plan starts a session
  Running --> Parked: p exits at waiting-for-user
  Parked --> Running: gogo go resumes the warm UUID
  Running --> AwaitingUAT: p exits at awaiting-uat
  AwaitingUAT --> Running: UAT feedback, re-accept, rerun
  AwaitingUAT --> Shipped: user runs gogo done
  Shipped --> Reaped: reap at ship
  Running --> Orphaned: crash or lost owner
  Orphaned --> Reaped: gogo sweep kills the tmux
  Reaped --> [*]

As-built structure — the session-manager types (cli/internal/orchestrator)

classDiagram
  class Session {
    +Root string
    +Slug string
    +Kind string
    +Attach bool
    +Takeover bool
    +LaunchOrResume() Outcome
    +Reap()
  }
  class SessionRunner {
    <<interface>>
    +Run(Invocation) RunResult
  }
  class ClaudeRunner {
    +Root string
    +Run(Invocation) RunResult
  }
  class AttachFn {
    <<func>>
    LaunchPersistent
  }
  class Registry {
    +Slug string
    +Persistent map~kind~PersistentSession
    +Sessions []SessionInfo
    +Get(kind) PersistentSession
    +Ensure(kind) PersistentSession
  }
  class PersistentSession {
    +Kind string
    +UUID string
    +Tmux string
    +Status string
    +CostUSD float64
  }
  class Owner {
    +PID int
    +UUID string
    +Tmux string
    +Host string
  }
  class Lock {
    +Path string
    +Owner Owner
    +Release() error
  }
  class LivenessFn {
    <<func>>
    PID signal-0 OR live gogo-* tmux
  }
  class Sweeper {
    +Root string
    +DryRun bool
    +Sweep() []string
  }
  class Invocation {
    +Kind string
    +Command string
    +SessionID string
    +Resume string
  }
  SessionRunner <|.. ClaudeRunner
  Session --> SessionRunner : Runner
  Session --> AttachFn : Attacher (--attach)
  Session --> Registry : Reg
  Session ..> Lock : Acquire / Release
  Session ..> LivenessFn : Live (cross-check)
  Session ..> Invocation : ResolveInvocation
  Registry "1" o-- "*" PersistentSession : per leg kind (go|plan)
  Registry "1" o-- "*" SessionInfo : append-only telemetry
  Lock *-- Owner
  Sweeper ..> Registry : reads + marks reaped
  Sweeper ..> LivenessFn : stale-lock heal
